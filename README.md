# 🛒 EGUS API — Sistema de Autoatendimento para Supermercados

Backend de um sistema de autoatendimento para supermercados, desenvolvido em **Java com Spring Boot**, com foco em automação de cadastro de produtos, gerenciamento de pedidos e integração com API externa para busca de informações por código de barras.

Projeto acadêmico com arquitetura inspirada em sistemas reais de mercado.

---

## 🎯 Objetivo do Projeto
Criar uma API REST capaz de:
- Automatizar o cadastro de produtos via código de barras
- Gerenciar pedidos de compra
- Simular o funcionamento de um sistema de autoatendimento (self-checkout)
- Trabalhar com regras de negócio e persistência de dados

---

## ⚙️ Funcionalidades
- Cadastro e consulta de produtos
- Leitura de código de barras
- Integração com API externa para obtenção automática de dados do produto
- Criação e gerenciamento de pedidos
- Atualização de status de pedidos
- Estrutura preparada para autenticação de usuários
- Persistência de dados em banco relacional

---

## 🧩 Arquitetura
- API REST seguindo padrão MVC
- Camada de serviços para regras de negócio
- Repositórios com Spring Data JPA
- Separação entre entidades, controllers e serviços
- Tratamento de exceções

---

## 🛠️ Tecnologias Utilizadas
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Git & GitHub
- Integração com API externa (REST)
